Arduino Electric Fan
This is Electric fan with remote controller.
This project was designed for end-of-term project by my team.
The first model of this project was just to add remote control system to orignial electric fan. We did it. But, professor provide feedback to add rotaitional angle control system.
We tried to add that function to our fan, but we couldn't find appropriate tool for our fan. So, we made new electric fan which includes remote control system and rotaitional angle control system.
It consists of Arduino Nano, L9110 Modules, HC-06 Bluetooth Module.
In this project, I was responsible for programming remote controller and Arduino sketch code and designing circuit.
